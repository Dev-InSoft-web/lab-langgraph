## Página 31

Desarrollo de Software Propio : El Facturador electrónico desarrolla su propio software para enviar sus facturas electrónicas a la DIAN . El desarrollo de Software debe pasar las pruebas técnicas de la DIAN para poder facturar electrónicamente . El software debe actualizarse constantemente para cumplir con las exigencias de la DIAN . Algunos desarrolladores bajo este esquema venden o licencian software para que los facturadores electrónicos lo utilicen como “Desarrollo de Software Propio”, bajo este esquema al momento de inscribir este software deben superar nuevamente las pruebas técnicas . El riesgo es que el desarrollador del software no tiene ningún vínculo ante la DIAN y por ende no tiene ninguna sanción si el software no funciona, la responsabilidad es 100 % de quien compra o licencia el software para facturar electrónicamente . Opciones para Facturar Electrónicamente: www.thefactoryhka.com

## Página 32

Proveedor Tecnológico : El facturador electrónico selecciona alguno de los Proveedores Tecnológicos aprobados por la DIAN, al momento del registro del facturador electrónico ante la DIAN, el facturador electrónico no debe pasar pruebas técnicas pues dichas pruebas técnicas ya las superó el proveedor tecnológico . El Proveedor Tecnológico tiene una altísima regulación por parte de la DIAN, lo cual lo obliga a prestar el servicio de manera correcta a quien contrató sus servicios so pena de enfrentar sanciones, multas o la pérdida de su habilitación como proveedor tecnológico .
