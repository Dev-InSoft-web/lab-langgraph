# Videos largos del canal

Subcarpetas por **año de publicación** (`ytdlp.upload_date`).

También existen hermanos en la raíz del corpus:

- `../shorts/{año}/` — YouTube Shorts
- `../streams/{año}/` — directos y replays

Nuevos contenidos: `npm run lab:yt:transcripts` (lista `/videos`, `/shorts` y `/streams`).
