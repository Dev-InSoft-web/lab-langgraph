## PatyIA — Actualizar NINSTRUCCION con nombres semánticos

Se actualiza la columna `NINSTRUCCION` de la tabla `INSTRUCCION` en
`AYUDASCP_IA` con nombres en español listos para usar como *label*
en el cliente (con tildes, mayúsculas y minúsculas correctas).

Los 13 tipos de instrucción quedan mapeados a:

| IINSTRUCCION | NINSTRUCCION |
|---|---|
| `ASESORIA_PERSONALIZADA` | Asesoría Personalizada |
| `COMERCIAL` | Comercial |
| `CONSULTA_NORMATIVA_NEGOCIO` | Consulta Normativa Negocio |
| `ERROR_ACCESO` | Error Acceso |
| `ERROR_CONFIGURACION` | Error Configuración |
| `ERROR_DIAN` | Error DIAN |
| `ERROR_TECNICO` | Error Técnico |
| `FUERA_DE_ALCANCE_TECNICO` | Fuera de Alcance Técnico |
| `INTERPRETACION_RESULTADO` | Interpretación Resultado |
| `PASO_A_PASO` | Paso a Paso |
| `REQUIERE_CONTEXTO` | Requiere Contexto |
| `SALUDO_OTRO` | Saludo Otro |
| `SOLICITUD_NO_PERMITIDA` | Solicitud No Permitida |
